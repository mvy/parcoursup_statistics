# Outils de traitement statistique des données officielles de Parcoursup

## Pré-requis

Pour utiliser ce script python vous devez disposer de :

- python 3+
- matplotlib
- pandas

Optionnel: spyder

La meilleure façon d'obtenir une distribution fonctionnelle pour est
l'installation de l'environnement 'Anaconda' qui devrait fournir tout ces
paquets (pandas n'y étant pas par défaut il me semble).

## Source des données 

Les données sont celles fournies par le [ministère de l'enseignement supérieur,
de la recherche et de l'innovation][1]
